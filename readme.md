# Video Repository
---
### Aims

The aim of this project is to create a Video Management Web interface that can be used to manage a video archive on the same or another machine.

#### Final Product goals

- Provide metrics for a video repository selected via the web interface
- Provide metrics for individual videos within the repository
- Provide a clean UI for navigating the information retrieved
- Provide a browser to select video files
- Allow users to edit indivdual files
- Allow users to mass edit files
- Provide graphs and visuals based on repository metrics
- Organise video files automatically based on different preferences

### Caution

This project only aims to be used on LAN and not to be exposed to the internet as such I would **HIGHLY** advise against doing so.
